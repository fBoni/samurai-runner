# samurai-runner

Samurai Runner is a 2D propotype game built in Godot 4.
The project serves as a study environment to learn movement, animation, camera behaviors, parallax background, and general game development workflows.

---

## Project Status
**Prototype / Learning Project**

Currently experimenting with character movement, camera settings, and basic level elements.

---

## Features
- 2D player character with basic horizontal movement
- Idle and running animations
- Camera with custom zoom (2.5x)
- Parallax background with horizontal looping
- Test environment for future level building

---

## How to run the project
1. Install **Godot 4.x**
2. Clone or download this repository
3. Open the project folder with Godot
4. Run the main scene
